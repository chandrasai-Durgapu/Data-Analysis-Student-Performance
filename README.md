# Data-Analysis-Student-Performance
Exploratory Data analysis on Student performance


## Clone the repository
```bash
https://github.com/chandrasai-Durgapu/Data-Analysis-Student-Performance.git
```
```bash
cd Data-Analysis-Student-Performance
```

## Create Virtual Environment
```bash
python -m venv data-analysis-student04
```
## Activate Virtual Environment
```bash
.\data-analysis-student04\Scripts\activate
```

## Install Dependencies
```bash
pip install -r requirements.txt
```
